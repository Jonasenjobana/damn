# 用户密码加盐与管理员 Seed 计划

## 1. 安装依赖
- 安装 `bcrypt` 用于密码加密
- 安装 `@types/bcrypt` 类型定义

## 2. 更新 User 实体
- 添加 `salt` 字段存储盐值
- 调整实体结构

## 3. 创建密码加密工具
- 创建 `password.service.ts` 提供密码加密和验证功能
- 生成随机盐值
- 使用盐值加密密码

## 4. 修改 UserService
- 更新 `create` 方法，在保存用户前对密码进行加盐加密

## 5. 创建管理员 Seed
- 创建 `seed.service.ts` 或 `seed.ts` 脚本
- 定义管理员账号信息
- 检查管理员是否已存在，避免重复创建

## 6. 集成 Seed
- 在应用启动时自动运行 seed
- 或提供单独的 seed 命令

## 7. 测试
- 测试用户注册功能
- 验证密码加密是否正常
- 验证管理员账号是否成功创建
