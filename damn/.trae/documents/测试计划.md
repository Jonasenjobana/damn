# 用户权限和 Session 测试计划

## 测试目标
1. 验证用户登录功能正常
2. 验证 JWT token 生成和验证
3. 验证需要认证的接口访问控制
4. 验证用户权限（管理员/普通用户）
5. 验证请求中的用户信息是否正确赋值

## 测试文件结构
```
src/
├── modules/
│   ├── auth/
│   │   └── auth.controller.spec.ts    # 认证模块测试
│   └── user/
│       └── user.controller.spec.ts    # 用户模块测试
test/
└── auth.e2e-spec.ts                    # 端到端认证测试
```

## 测试内容

### 1. AuthController 单元测试
- 测试登录接口返回 JWT token
- 测试错误密码返回 401
- 测试不存在用户返回 401

### 2. UserController 单元测试
- 测试未登录访问受保护接口返回 401
- 测试登录后访问受保护接口成功
- 测试请求中用户信息正确

### 3. 端到端测试
- 完整流程：注册 → 登录 → 访问受保护接口
- 验证管理员权限
- 验证普通用户权限
